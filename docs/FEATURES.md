# Features Overview

## Workspace Platform
- Infinite canvas with persistent node layout
- Workspace-scoped files and SQLite storage
- Plugin state persistence per workspace
- Optional protected workspace access

## Developer Tooling
- HTTP Tester
- SQL Viewer
- File Manager (tabs, quick open, typed previews)
- Terminal websocket shell
- FTP server/client tools

## Data & Content Tools
- JSON/YAML/XML/CSV processors
- Diff and Regex tooling
- Markdown and logs parsing viewers

## Security Utilities
- Hashing, HMAC, encryption workflows
- Base64 and JWT analysis tools
- Password and UUID generation

## System Operations
- System metrics/info
- Disk and task views
- Settings and secrets handling
- Activity/logs visibility

## Extensibility
- iframe-based plugin architecture
- plugin API helper (`pluginFetch`)
- route modularization in `server/api/*`
