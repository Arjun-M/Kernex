# Plugin Catalog

## Productivity Utilities
- Website Extractor: extract structured content from a URL.
- Note: markdown-based quick notes on the canvas.

## HTTP / API
- HTTP Tester: request builder, collections, environments, history, response viewer.

## System & Environment
- Terminal: interactive shell over websocket.
- FTP Client: connect to external FTP servers.
- FTP Info: connection info for built-in FTP service.

## Database Tools
- SQL Viewer: inspect DB info, tables, schema, rows, and run SQL queries.

## Data Formats
- JSON Tool: pretty/minify/validate JSON.
- YAML Tool: YAML/JSON conversions.
- CSV Viewer: parse and table-preview CSV.
- XML Tool: format and validate XML.

## Text Utilities
- Diff Tool: old/new text diff view.
- Regex Tool: pattern matching and groups preview.
- Markdown: rendered preview.
- Log Viewer: parsed logs with level filters.

## Cryptography / Security
- Hash Generator: md5/sha/bcrypt flows.
- Base64 Tool: encode/decode.
- JWT Decoder: parse token sections.
- UUID Generator: batch UUID generation.
- Password Generator: configurable secure password generation.
- HMAC Tool: signature generation with selected algorithm.
- Encryption: AES encrypt/decrypt options.

## URL & Routing
- Short URLs: CRUD for short links and redirect handling.

## File/Workspace Tools
- File Manager: tree, tabs, quick-open, previews, safe text mode.

## Plugin Selection Guidance
- Need API testing: use HTTP Tester
- Need DB inspection: use SQL Viewer
- Need content transforms: use JSON/YAML/XML/CSV tools
- Need security tooling: crypto suite plugins
- Need filesystem operations: File Manager
